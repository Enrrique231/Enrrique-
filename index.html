¿Que es un protocolos Http? Es el nombre de un protocolo el cual nos permite realizar una petición de datos y recursos, como pueden ser documentos HTML ¿Que es un query String? Las Query String o cadenas de consultas es un término que se utiliza para hacer referencia a una interacción con una base de dato ¿Que es un verbos Http? Se utiliza para recuperar información desde un recurso ¿que es un DNS? El sistema de nombres de dominio (DNS) es el directorio telefónico de Internet. Las personas acceden a la información en línea a través de nombres de dominio como nytimes ¿Que es un CDN? es una red de servidores interconectados que acelera la carga de las páginas web para las aplicaciones que tienen un uso intensivo de datos
